## Project Overview

This is a **Docusaurus v3** documentation website for the "Introduzione ai Database, Data Analytics e AI" course, a partnership between FINANZA.TECH and MCG Consulting. All content is written in **Italian**.

The site is deployed at: https://mcg-consulting2026.vercel.app/

## Commands

Run from the repo root (`/Users/davidedantonio/Documents/MCG_Consulting_2026/repos/website`):

```bash
yarn install        # Install dependencies
yarn start          # Start dev server with hot reload
yarn build          # Build static site to /build
yarn serve          # Preview production build locally
yarn typecheck      # TypeScript validation
yarn clear          # Clear Docusaurus cache (use when things look stale)
yarn deploy         # Deploy to GitHub Pages
```

Requires **Node.js >= 20**.

## Architecture

### Framework
Docusaurus 3.9.1 with React 19 and TypeScript. Sidebars are auto-generated from the `/docs` directory structure via `sidebars.ts`. MDX is supported for interactive content.

### Content Structure
```
docs/
├── intro.mdx                     # Course intro page
├── intro-to-database/            # Database fundamentals section
└── nodejs/                       # Node.js section (this directory)
    ├── _category_.json           # Sidebar label/position for this section
    ├── NN-slug.md                # Lessons (numbered for ordering)
    └── img/ch-N/                 # Chapter images
```

The numeric prefix on filenames (`01-`, `02-`, etc.) controls sidebar ordering.

### Adding/Editing Lessons
- New `.md` or `.mdx` files in `docs/nodejs/` are automatically picked up by the sidebar.
- Add images under `img/ch-N/` and reference them with relative paths.
- Use front matter to set `sidebar_position`, `title`, and `description`.
- Mermaid diagrams are enabled — use ` ```mermaid ` fenced blocks.
- Live code blocks are enabled via `@docusaurus/theme-live-codeblock`.

### Key Config Files
- `docusaurus.config.ts` — site metadata, navbar, footer, plugins, theme config
- `sidebars.ts` — sidebar generation (currently `autogenerated` from `/docs`)
- `src/components/` — custom React components used in MDX pages
